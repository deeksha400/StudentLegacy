# CrediVault

CrediVault is a premium AI-powered university achievement and certificate management platform built with React, TypeScript, Tailwind CSS, Framer Motion, and a Spring Boot backend scaffold.

## Included experience
- Landing page
- Login / Register view
- Student dashboard
- Digital achievement passport
- Certificate upload and AI verification UI
- Campus achievement wall
- Faculty verification dashboard
- Admin analytics dashboard
- Resume generator page

## Run the frontend
```bash
cd /workspaces/StudentLegacy
npm install
npm run dev
```

Open http://localhost:3000/

## Run the backend
```bash
cd /workspaces/StudentLegacy/backend
mvn spring-boot:run
```

API health check:
```bash
curl http://localhost:8080/api/health
```
